# vfox-alertmanager

[![CI](https://github.com/jbox-web/vfox-alertmanager/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-alertmanager/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [alertmanager](https://github.com/prometheus/alertmanager)

## Install

```
mise plugin install alertmanager https://github.com/jbox-web/vfox-alertmanager.git
```
